# MIS Bot

Автоматизация работы с медицинской информационной системой (МИС) с использованием OCR, шаблонного распознавания и управления интерфейсом.

## Возможности

### Работа с пациентами
- Поиск пациента по ФИО
- Проверка даты рождения
- Работа с открытой карточкой пациента
- Ручной и автоматический режим

### OCR
- OCR из изображения
- OCR из выделенной области экрана
- Двухшаговый OCR:
  - ФИО
  - дата рождения
- Массовый импорт пациентов
- Двойной проход OCR даты рождения
- Очистка ФИО (кириллица)

### Автоматизация МИС
- Выбор окна МИС
- Выбор шаблона исследования
- Автоматическое заполнение даты исследования
- Работа с услугами
- Работа с историей болезни
- Подпись исследования

### Интерфейс
- Progress окно
- Отображение текущего пациента
- Пауза выполнения
- Отмена выполнения
- Always-on-top окно

### Надежность
- Checkpoint система
- Fail-safe защита мыши
- Интерактивная калибровка
- Runtime-коррекция кликов
- Восстановление сессии

## Структура проекта

```text
config/     - настройки и координаты
gui/        - интерфейс

ocr/        - OCR
project/    - основная логика
services/   - сервисные функции
```

## Установка

### Клонировать репозиторий

```bash
git clone https://github.com/Stasikent/MIS-Bot.git
cd MIS-Bot
```

### Установка зависимостей

```bash
pip install -r project/requirements-offline.txt
```

---

## Запуск

```bash
python main.py
```

или:

```bash
run_bot.bat
```

---

## Настройки

Локальные настройки не входят в репозиторий.

Используйте:

```text
config/settings.example.json
```

и создайте:

```text
config/settings.json
```

---

## Важно

В репозиторий не входят:

- локальные пути
- пароли
- настройки пользователя
- временные файлы
- логи
